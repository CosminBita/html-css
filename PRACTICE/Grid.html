In lecțiile anterioare au fost prezentate tehnicile de bază pentru configurarea lui CSS Grid. Astfel, au fost procesate noţiunile de coloane și rânduri, interval, aranjarea elementelor în funcție de numerele liniilor, zonele și zonele numite... Însă, până acum nu s-a discutat despre alinierea elementelor Grid. Prin urmare, lecția de faţă va fi dedicată acestei tematici.

Pentru a înțelege corect logica care stă la baza alinierii elementelor Grid, este necesar să se înțeleagă conceptul de axe ale sistemului Grid. Având în vedere că este un sistem bidimensional, CSS Grid are două axe: axa block (verticală), respectiv axa coloanelor; axa inline (orizontală), respectiv axa rândurilor.
